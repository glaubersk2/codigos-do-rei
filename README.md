# 🦁 Códigos do Rei - Portal Institucional

Este repositório contém o código-fonte do portal institucional do **CÓDIGOS DO REI**, uma iniciativa dedicada à transformação cultural através da fé e educação. Localizado em São Gonçalo, o projeto atua com impacto evangelístico, social e cultural, promovendo a cultura cristã na sociedade.

## 🎯 Nossa Missão

"Cooperar com a transformação de uma cidade evangélica em uma cidade de cultura cristã inserida na sociedade."

## 🚀 Tecnologias Utilizadas

O projeto foi desenvolvido com uma stack moderna de front-end para garantir performance, responsividade e uma experiência de usuário premium:

- **React 19** - Biblioteca para interfaces de usuário.
- **Vite** - Build tool ultrarrápido.
- **Tailwind CSS 4.0** - Framework utilitário para estilização moderna e responsiva.
- **Motion (Framer Motion)** - Biblioteca de animações fluidas e interações.
- **Lucide-React** - Conjunto de ícones minimalistas.
- **TypeScript** - Tipagem estática para maior segurança no código.

## 🛠️ Como rodar o projeto localmente

### Pré-requisitos
- Node.js (v18 ou superior)
- Gerenciador de pacotes (npm)

### Passo a passo
1. Clone o repositório em sua máquina:
   ```bash
   git clone https://github.com/seu-usuario/codigos-do-rei-home-v1.git
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
4. Acesse o projeto em seu navegador:
   `http://localhost:5173`

## 📁 Estrutura do Projeto

- `src/App.tsx`: Componente principal com as seções do site (Hero, Sobre, Projetos e Doações).
- `src/constants.ts`: Arquivo central de conteúdo para fácil manutenção de textos e links.
- `public/`: Pasta destinada a ativos estáticos como logos e vídeos.
